# Login Page

A React sign-in page matching the design with social provider logos and Firebase configuration banner.

## Structure

Each component has its own JSX and CSS file:

- **SignInHeader** – Title and subtitle
- **SocialSignIn** – Google, Microsoft, Facebook, and Apple sign-in logos with links (above the banner)
- **InfoBanner** – Yellow Firebase configuration notice
- **SignInForm** – Email, password, forgot password link, Sign in button
- **SignUpLink** – Don't have an account? Sign up
- **SignInCard** – Main container assembling all components

## Social Sign-in Links

The social logos link to:

- Google: `https://accounts.google.com`
- Microsoft: `https://login.microsoftonline.com`
- Facebook: `https://www.facebook.com`
- Apple: `https://appleid.apple.com`

These are placeholders. Replace with your app’s OAuth URLs when you configure Firebase.

## Run

```bash
npm install
npm run dev
```
